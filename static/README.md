# Intelligent Career Guidance System

A Machine Learning based web application that predicts suitable career paths for students based on their skills, interests, and technical strengths.

## Technologies Used
- Python
- Flask
- Machine Learning
- NumPy
- HTML/CSS

## Features
- Career prediction system
- Skill-based recommendation
- Interactive web interface
- ML model integration

## Tools & Libraries
- Flask
- Scikit-learn
- NumPy
- Pickle

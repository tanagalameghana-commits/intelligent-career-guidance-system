# Intelligent Career Guidance System

A Machine Learning based web application that predicts suitable career paths for students based on their technical skills, interests, and strengths.

## Overview

This project helps students identify suitable career domains using Machine Learning algorithms. Users provide their skill ratings and the system predicts the most relevant career options.

The application is developed using Flask for the web interface and a trained Machine Learning model for prediction.

---

## Features

- Career prediction based on student skills
- Interactive web interface
- Machine Learning based recommendations
- Skill analysis system
- Multiple career role suggestions

---

## Technologies Used

- Flask
- Machine Learning
- NumPy
